# Vertex Generative AI Overview

Google Cloud provides a portfolio of enterprise-grade AI services that can scale with product and business needs.

## Key Notes

- Chirp is Google Cloud's speech-to-text model family for speech AI use cases.
- Vertex AI Model Garden provides access to Google, third-party, and open source foundation models.
- Vertex AI Search supports enterprise search experiences that can be used out of the box or customized.
- Vertex AI Agent Builder helps teams build search, recommendation, and conversational agent experiences.
- PaLM supports text-focused language model use cases.
- Gemini supports multimodal text, image, video, and audio use cases.
- Vertex AI Studio provides a browser-based environment for experimenting with generative AI.
